Best Practices للـ .NET Project
  • مفيش بزنس لوجيك يخرج بره Service
  • دايما لازم نرجع Error Messages عشان الفرونت أو المستخدم يعرف ايه المشكلة
  • اي DTO متشابه لازم يحتوي على نفس اسماء ال Properties
  • اي صورة فى المشروع هيبقي اسمها ImageUrl
  • نحاول دايما نطبق SOLID و Design Patterns على الشغل
  • لازم نتأكد من ان الكود اللى اتكتب الأداء بتاعه كويس ومنعديش سطر كود بدون مراجعة
  • قبل عمل أي شئ لازم نكتب المطلوب فى كل Method بالخطوات
  • نشتغل بال Start Transaction و Rollback
  • دايما ال Repositories ملهاش علاقة ب Business Roles، يعني ملقيش فيها if ولا switch
  • ال Query Repository لازم دايما ترجع DTO
  • هنعمل Mapper Extension Methods وهتضاف داخل كل Command أو Query فى ال CQRS
  • ال Base URL اللى بيضاف للصورة أو الملف هيبفي من خلال ال CQRS Query
  • لازم دايما نحفظ التاريخ والوقت UTC ونرجعه بال Local Time Zone بتاع المستخدم
  • دايما لما نعمل أي حاجه تبقي ب Tenant و Project
  • لو عاوز ترجع الاسم أو أي حاجة بناءً على اللغة، اعمل دا فى ال Handler
  • لو عاوز ترجع الوقت والتاريخ حسب المنطقة الخاصة بالمستخدم، اعمل دا فى ال Handler
  • أي ميثود بتاخد أكتر من برامتر واحد لازم يتعمل ليها request dto
